Dheez's Review


Left 4 Dead 2 atau biasa dikenal dengan L4D2 adalah game buatan Valve yang dirilis pada 17 November 2009. Game ini memang keluaran lama, tapi jangan diremehkan. Game keluaran tahun 2009 ini masih dimainkan oleh ribuan orang.

L4D2 merupakan game first-person shooter dimana kita akan melawan dan menembaki zombie beserta mutant yang ada. Dalam 1 tim terdapat 4 pemain, kamu dapat bermain sendiri bersama 3 bot atau bermain dengan teman. Yah simpelnya, kamu harus bertahan untuk menuju safe house hingga akhir dari permainan sambil bertahan hidup dari para zombie dan mutant.

Untuk game keluaran tahun 2009, saya merekomendasikan game ini apalagi ketika diskon. Tidak perlu merasa ragu untuk membeli game ini, karena game ini masih seru untuk dimainkan apalagi jika kalian bermain multiplayer yang akan membuat game semakin sulit walaupun kalian memilih kesulitan normal.

Game ini dapat kalian beli pada Steam dengan harga Rp 90.999. Untuk informasi harga yang lebih lanjut, kalian dapat mengecek disini, Price History Left 4 Dead 2.
