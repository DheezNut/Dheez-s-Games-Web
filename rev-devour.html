Dheez's Review


Devour merupakan game horror buatan Straight Back Games yang dirilis pada 28 Januari 2021. Karena saya penakut, saya akui game ini memang agak mengerikan untuk saya dan untuk jumpscare mungkin akan mengejutkan diawal, tetapi kelamaan kalian akan terbiasa.

Untuk storyline pada Devour sendiri akan berbeda-beda setiap map, tetapi inti dari gamenya sendiri adalah kita diharuskan untuk menghentikan seorang anggota kultus yang telah dirasuki oleh iblis yaitu Azazel. Jadi dalam game ini kita harus mencari cara bagaimana untuk menghentikannya dan bertahan dari ancaman musuh yang memburu kita.

Menurut saya, game ini lebih asik jika dimainkan bersama teman. Apalagi jika kalian merasa susah ketika bermain sendiri, mungkin bisa mengajak teman kalian untuk bermain dan mengalahkan iblis itu bersama.

Game ini dapat kalian beli pada Steam dengan harga Rp 90.999. Untuk informasi harga yang lebih lanjut, kalian dapat mengecek disini, Price History Devour.
