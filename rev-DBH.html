Dheez's Review


Detroit: Become Human merupakan game buatan Quantic Dream yang dirilis pada 18 Juni 2020. Game ini merupakan sebuah game naratif dimana cerita dari game ini tergantung oleh kita sendiri. Bisa dikatakan game Detroit: Become Human merupakan tipe game choices matter, yang berarti pilihan kita saat memainkan game ini akan bergantung untuk kedepannya. Tentu, game tipe seperti ini memiliki banyak sekali ending.

Intinya, Detroit: Become Human menceritakan pada tahun 2038 telah banyak perkembangan teknologi, salah satunya terdapat robot atau disebut sebagai android dalam game yang memiliki kecerdasan bahkan bersikap layaknya manusia tetapi lebih sempurna, para android ini diciptakan untuk melayani manusia. Dalam game ini, konflik utamanya adalah dimana beberapa android tersebut menyadari bahwa mereka merasa tidak adil harus melayani atau patuh kepada para manusia dan menginginkan keadilan atau kesetaraan antara android dan manusia. Pada game, terdapat 3 android yang akan kita mainkan ketiganya yaitu Connor, Marcus dan Kara. Tentu, ketiga karakter ini akan saling terhubung dalam ceritanya dan tergantung dari pilihan kita, salah satu atau ketiga karakter ini bisa saja berakhir mati.

Saya sangat merekomendasikan game ini jika kalian adalah tipe yang fokus pada storyline dalam game. Selain itu, game ini juga memberi grafis yang memanjakan mata. Namun, game ini memiliki masalah untuk kalian user komputer atau laptop. Karena ketika kalian pertama kali memainkan game ini, ada kemungkinan kalian akan menghadapi masalah FPS drop atau stuttering, tidak peduli spesifikasi kalian sebagus apa. Tetapi tidak perlu khawatir, karena kalian dapat mencari solusinya di internet jika kalian mendapat masalah ini.

Game ini dapat kalian beli pada Steam dengan harga Rp 420.000. Untuk informasi harga yang lebih lanjut, kalian dapat mengecek disini, Price History Detroit: Become Human.
