Dheez's Review


Content Warning adalah sebuah game indie horror yang rilis pada 1 April 2024. Content Warning membawakan sebuah gameplay horror walau menurut saya tidak yang benar-benar horror, tapi saya akui ada beberapa saat dimana saya merasa sedikit takut.

Tidak ada storyline untuk game ini, kamu hanya berperan sebagai seseorang yang ingin menjadi terkenal dengan merekam kejadian horror pada tempat yang kamu kunjungi. Semakin banyak monster atau hal mencekam yang kamu rekam, maka views dan profitmu semakin tinggi. Saya sedikit bingung bagaimana ingin mengatakan ini, tetapi pada Content Warning, kamu memiliki batas 3 hari untuk mencapai views tertentu. Jika tercapai maka kamu akan melanjutkan game dengan menuju hari selanjutnya, jika tidak maka kamu akan mengulang dari hari pertama.

Menurut saya, game Content Warning ini lebih asik untuk dimainkan bersama teman dibandingkan bermain sendiri. Untuk gamenya sendiri, menurut saya sudah bagus dan seru-seru saja. Hanya, menurut saya kurangnya dari game ini adalah kurangnya informasi mengenai sesuatu seperti item, monster, dan lainnya didalam game sehingga mengharuskan kamu untuk mencari informasi mengenai itu diluar gamenya.

Game ini dapat kalian beli pada Steam dengan harga Rp 74.000. Untuk informasi harga yang lebih lanjut, kalian dapat mengecek disini, Price History Content Warning.
