Dheez's Review


Inscryption merupakan game buatan Daniel Mullins Games yang dirilis pada 19 Oktober 2021. Game ini simpelnya adalah permainan kartu tetapi terdapat sedikit elemen horror dan adventurenya. Saya bukanlah tipe pemain kartu apalagi strategi maka saya sedikit ragu terhadap game ini sampai saya mencoba demo dari game ini. Saya bisa bilang, wow... Karena game ini membuat saya teradiksi atau candu untuk memainkannya walau ini hanya game kartu.

Saya tak bisa bilang banyak untuk storyline karena lebih baik jika kalian mainkan sendiri. Inti dari game Inscryption sendiri ya kalian bermain kartu, menyusun strategi baik dari ketika bermain kartu atau mengumpulkan item, mengadu nasib ketika melakukan sesuatu, mengumpulkan kartu atau build deck kartu yang baik, jika kalah mengulang dari awal. Tetapi, tidak hanya begitu saja karena game ini memiliki 3 chapter dan saya tidak akan mengatakan detailnya.

Saya sangat merekomendasikan game ini untuk kalian yang menyukai game kartu. Tetapi untuk kalian yang seperti saya yang tidak terlalu atau tidak pernah bermain kartu dan masih ragu, kalian dapat mencoba demo dari game ini di website GOG.

Game ini dapat kalian beli pada Steam dengan harga Rp 119.999. Untuk informasi harga yang lebih lanjut, kalian dapat mengecek disini, Price History Inscryption.
